# Ground Truth 对比可视化功能总结

## 修改概述
已成功为 `inference_sd_non_connection_shallow.py` 推理脚本添加了与 Ground Truth (GT) 的对比功能。

## 主要功能

### 1. **Ground Truth 数据加载** (第57-68行)
- 从测试数据集中自动加载第一个样本作为 Ground Truth
- 支持当数据集不可用时的优雅降级处理
- 加载后打印 GT 张量的形状以供验证

### 2. **对比动画可视化** (第330-385行)
当 Ground Truth 可用时，生成三列并排展示的动画：
- **左列**: Ground Truth 场
- **中列**: 模型生成的预测场  
- **右列**: 绝对误差 (|Pred - GT|)

**特性**：
- 统一的颜色范围（使用 GT 和 Pred 的合并min/max）
- 独立的colorbar便于理解每列的值范围
- 动画文件保存为 `comparison_trajectory.gif`

### 3. **频域能量谱对比** (第402-565行)
如果有 GT 可用，生成 2×2 的频域分析图：

| 位置 | 内容 | 说明 |
|------|------|------|
| (0,0) | 时间轴能量谱 | 对比生成的和GT的频谱 |
| (0,1) | 时间轴误差谱 | 频域能量差异 |
| (1,0) | 空间能量谱 (t=0) | 对比初始时刻的空间频谱 |
| (1,1) | 空间误差谱 (t=0) | 空间频域能量错误 |

**可视化特点**：
- 生成的结果用实线 ○ 标记
- GT 用虚线 □ 标记，透明度70%
- 对数坐标便于观察多个数量级
- 分别显示各通道（涡度/高度）的谱

### 4. **定量误差指标计算** (第568-633行)
计算并保存详细的误差指标到 `error_metrics.txt`：

**全局误差**：
- MSE (L2²)
- RMSE (L2)
- MAE (L1) 
- Max Error (L∞)

**逐通道误差**：
- 涡度 (Vorticity): RMSE, MAE, MSE
- 高度 (Height): RMSE, MAE, MSE
- 其他通道（如果有）

**时间演化**：
- 显示每个时间步的 RMSE
- 帮助理解误差如何随时间发展

## 输出文件

推理运行后会生成：

1. **comparison_trajectory.gif** - 三列并排的对比动画
2. **frequency_energy_spectrum.png** - 频域对比图（2×2 或 1×2）
3. **error_metrics.txt** - 详细的数值误差指标

## 使用方式

保持原有的使用方式不变：
```bash
python -m scripts.inference_sd_non_connection_shallow config_shallow_Gabor.yml
```

脚本会自动：
1. 检测是否有有效的 Ground Truth 数据可用
2. 如果可用，生成对比可视化
3. 如果不可用，使用原始单一视图

## 代码设计特点

- **向后兼容性**: 当 GT 不可用时，脚本按原方式运行
- **模块化设计**: GT 加载、对比可视化、指标计算相互独立
- **灵活的构型**: 支持不同数量的通道和数据维度
- **有针对性的错误处理**: 详细的日志输出便于调试
- **强大的可视化**: 自动计数器防止覆盖已有文件

## 实现细节

### 颜色映射
- 预测和GT均使用 `RdBu_r` 色图（红蓝反转）
- 误差使用 `hot` 色图（热力）
- 频谱使用 `viridis`（默认）

### 频域分析方法
1. **时间轴FFT**: 沿时间维度的1D FFT
2. **空间FFT**: 初始时刻通过插值到规则网格进行2D FFT
3. **径向平均**: 计算各波数的能量谱

### 误差度量
- 使用 NumPy 向量化操作确保效率
- 支持带 NaN 的插值后的网格
- 全面覆盖L1、L2、L∞范数

## 配置项

脚本尊重现有配置，新增功能无需额外配置参数。所有参数从config文件或代码默认值获取：
- `sampling_mode`: 采样模式（1-step或多步）
- `num_sampling_steps`: 采样步数
- `t_max`, `t_min`: 时间步长范围
- 等等...

## 依赖

新增功能仅使用现有依赖：
- `matplotlib` - 动画和绘图
- `numpy` - 数值计算
- `scipy` - 网格插值
- PyTorch - 张量处理

无需添加新的外部库。
